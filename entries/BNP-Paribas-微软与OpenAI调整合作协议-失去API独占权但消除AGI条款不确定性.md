# BNP — 微软与OpenAI调整合作协议

## 摘要
BNP分析微软与OpenAI修订版合作框架。核心变化：Azure失去OpenAI API的stateless exclusivity（结构性）、消除AGI条款使OpenAI收入继续计入微软、微软在OpenAI的股权投资回报更清晰。BNP认为整体影响中性偏正面——排除AGI不确定性是长期利好，但Azure API独占权丧失是竞争劣势。

## 关键要点
- OpenAI stateless API独占权从Azure移除（其他云可分发）
- AGI条款消除 → OpenAI收入可持续计入微软
- 微软在OpenAI的股权投资回报机制更清晰
- Azure结构性优势减弱，但Copilot等自有AI产品不受影响
