# BNP — 微软多线推进，Copilot拉动增长

## 摘要
BNP深入分析微软的两大核心业务。M365 Commercial Cloud和Azure合计占收入60%，Azure是主要增长引擎。Copilot在M365/Dynamics/GitHub等产品的渗透率持续提升，开始贡献实质性增量收入。微软在AI货币化方面进展领先同行。

## 关键要点
- M365 Cloud + Azure = 60%收入
- Copilot渗透率持续提升
- GitHub Copilot已大规模商用
- AI货币化领先同行
