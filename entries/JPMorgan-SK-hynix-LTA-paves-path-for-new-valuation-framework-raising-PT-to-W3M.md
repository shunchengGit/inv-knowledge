# JPMorgan: SK hynix LTA Paves Path for New Valuation Framework

**报告日期**: 2026-05-17 | **券商**: JPMorgan | **分析师**: Jay Kwon, Sangsik Lee, Neelay Y Kamath  
**评级**: Overweight (维持) | **目标价**: KRW 3,000,000（上调自 KRW 1,800,000）| **涨幅**: +67%

## 核心观点

### LTA 开启估值框架结构性转变
- SK Hynix 正处于与 CSP（云服务提供商）客户进行 LTA 讨论的关键拐点
- LTA 将推动存储器业务模式从"周期性"转向"结构性/长期性"
- 存储器供需失衡短期内难以缓解，存储器厂商和 CSP 达成 LTA 的概率很高
- LTA 将提升存储器盈利质量，成为估值重估的核心驱动因素

### 从 P/B 转向 P/E 估值框架
- JPMorgan 认为 P/E 估值框架更合理，使用 8x FY26-27E EPS
- 历史对比：2024 年以来平均 FTM P/E 仅 3x，当前约 6x
- 5 倍 EPS 增长后（FY26E），预计 FY28 前 EPS CAGR 仍达 31%
- Dec-26 目标价 KRW 3,000,000 = 8x 平均 FY26-27E EPS

### 盈利假设大幅上调
- FY26-28E EPS 上调 9-20%，反映更强的 DRAM/NAND/HBM 定价预期
- DRAM ASP 展望从低-40% q-q 上调至低-50% q-q
- 服务器 DRAM（AI 向 CPU 扩散）和移动 DRAM（LPDDR5X 竞争）共同驱动价格
- 三星罢工也对定价情绪产生积极影响
- HBM ASP 谈判预计对存储器厂商有利，明年 ASP 增长加速

### 股东回报与现金生成
- 预计 2025-27E 累计 FCF 达 KRW 240tn 用于股东回报
- 2026E/27E TSR 收益率达 5%/12%
- 官方股东回报政策：TSR = FCF 的 50%（三年累计基准）、25% 固定 DPS 提升、资本强度目标中 30%+、资产负债改善计划

### 核心催化剂与风险
**催化剂**: LTA 量/价协议披露 + 股东回报执行 + EPS 持续上修  
**风险**: LTA 谈判停滞、AI 需求放缓、地缘政治风险、技术突破减少 AI 机架内存需求、存储器供给超预期
